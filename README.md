# 🟢 Simulación de Círculos con Canvas, JavaScript y Bootstrap

## 📌 Descripción del proyecto

Este proyecto consiste en una aplicación web interactiva desarrollada con **HTML5 Canvas**, **JavaScript** y **Bootstrap 5**, cuyo objetivo es visualizar y comparar distintos comportamientos físicos simulados en múltiples conjuntos de círculos.

La aplicación permite observar en tiempo real tres escenarios distintos:

- Movimiento de círculos con rebote en bordes  
- Detección de colisiones entre círculos  
- Rebote dinámico entre círculos tras colisionar  

Además, el usuario puede controlar dinámicamente la cantidad de elementos mediante un **desplazador interactivo**, provocando la actualización inmediata de las simulaciones.

---

## 🎯 Objetivo

Desarrollar una aplicación web interactiva que permita analizar el comportamiento de múltiples objetos en movimiento utilizando lógica de simulación, detección de colisiones y renderizado en tiempo real con Canvas.

---

## 🧠 Conceptos aplicados

- Programación orientada a objetos (OOP) en JavaScript  
- Manipulación del elemento `<canvas>`  
- Animaciones con `requestAnimationFrame`  
- Detección de colisiones (distancia euclidiana)  
- Interacción en tiempo real con el DOM  
- Diseño responsivo con Bootstrap  
- Estilos modernos con Glassmorphism  

---

## ⚙️ Funcionalidades

✔ Visualización de tres simulaciones simultáneas  
✔ Control dinámico del número de círculos  
✔ Animaciones fluidas en tiempo real  
✔ Rebote contra bordes del canvas  
✔ Detección de colisiones por traslape  
✔ Intercambio de velocidad en rebotes  
✔ Cambio de color al colisionar  
✔ Interfaz moderna y responsiva
